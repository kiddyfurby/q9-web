Q9 Extension

Manual Installation Guide

Download Extension (.zip)
1

Extract the ZIP

Unzip the shared q9-extension.zip file to a folder on your computer.

2

Open Extensions Page

In Chrome, type chrome://extensions in the address bar and press Enter.

3

Enable Developer Mode

Toggle the Developer mode switch in the top right corner of the page.

4

Load Extension

Click Load unpacked and select the dist folder.

5

Activate Data

Once installed, open the extension options or click "Activate" in the widget to upload your Q9.IM_ file.
